Couleur

Titres en couleur

  1. Écrire un fichier style.css, dans le lequel :

Les titres de niveau 1 seront en rouge

Les titres de niveau 2 seront en vert

Le texte des paragraphes en bleu

  1. Copier dans le dossier du fichier style.css, les fichiers « informations_sup.html » et « listes_a_puces.html »

  2. Modifier le fichier listes_a_puces.html pour qu'il soit lié au fichier CSS.

  3. Ouvrir les deux fichiers html dans le navigateur

  4. Modifier, maintenant le fichier « informations_sup.html » pour qu'il soit lui aussi lié au fichier « style.css »

  5. Ré-ouvrir les deux fichiers dans le navigateur.


Paragraphe en couleur

  1. Placer le fichier « paragraphes.html » dans le répertoire de « style.css »

  2. Modifier le fichier CSS et le fichier « paragraphes.html » pour que les 2ème et 4ème paragraphes soient en orange.
